# Project: Unknown

## Description
Project: Unknown is an isometric survival simulator game about re-creating life from virtually nothing while upgrading your bunker and world to survive. Awaking in a bunker, while knowing very little, you have to figure out the unknown world. Throughout the game, you learn about how the world came to be, create plants and crops, mine minerals and materials, and explore this world. Try not to get infected. It seems to...mess with your memory.

---

## Features - Completed
- 8-Direction Player Movement + Animation
- Working Inventory System With Drag-And-Drop
- Procedurally Generated World + Multi-Layer height
